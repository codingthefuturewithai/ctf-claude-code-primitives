# CTF Claude Code Primitives

Personal Claude Code primitives for [Coding the Future with AI](https://github.com/codingthefuturewithai).

## Available Skills

| Skill | Description |
|-------|-------------|
| `ctf-claude-code-primitives:ctfai-brand` | Apply Coding the Future with AI brand styling to PDFs, websites, React themes, and other artifacts |

## Installation

Install skills directly from GitHub using `ai-agent-skills`:

```bash
npx ai-agent-skills install codingthefuturewithai/ctf-claude-code-primitives/ctfai-brand --agent claude
```

After installing, restart Claude Code. The skill will activate automatically when creating branded content or can be invoked with `/ctf-claude-code-primitives:ctfai-brand`.
